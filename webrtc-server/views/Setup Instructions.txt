How to Setup WebRTC Server
--------------------------

To Install and Run:
1) Install NodeJS and NPM
2) Copy webRTC-server files
3) In the WebRTC-server folder, run command "npm install" which will download and install all dependencies
3-B) Run “sudo npm install -g forever”
4) In the WebRTC Server folder, run command "npm start" which will start the server and keep it running
5) Enjoy!

To Stop:
1) In the WebRTC Server folder, run command "npm stop" which will stop the running server
